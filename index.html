Compiler::Lexer

- Fast Lexical Analyzer for Perl5 -

Features

  • Written by C++ [XS]
  • Perfect hashing for reserved keywords
  • Using memory pool for token objects
  • Readable Code (without yacc/bison)

PLAY GROUND

- Tokenize your perl5 code dynamically and display results to right pane -

emacs vim

>>

SYNOPSIS

use Compiler::Lexer; my $tokens = Compiler::Lexer->new->tokenize('print "hello world\n"'); print $tokens->[0]->data; #=> 'print' print $tokens->[1]->data; #=> 'hello world\n'

INSTALL

cpanm Compiler::Lexer
